# Portfolio Website

Static portfolio website for Bhavinkumar Umatiya.

## Files

- `index.html` contains the page structure.
- `styles.css` contains the visual design and responsive layout.
- `script.js` contains the project data and project card rendering.

## Update project links

Open `script.js` and fill the `github` and `demo` fields for each project.

## Run locally

Because this is a static site, you can open `index.html` directly in a browser.
If you want a local server, use any simple static server extension or tool.
